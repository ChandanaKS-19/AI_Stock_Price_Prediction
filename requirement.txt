# 🚀 AI-Based Stock Price Prediction App

## 📊 Overview
This is a real-time stock prediction web application built using Machine Learning and Streamlit. Users can select companies or enter custom stock symbols to visualize stock trends and predict future prices.

## ✨ Features
- 📈 Real-time stock data visualization
- 🔍 Company selection (TCS, Infosys, Reliance, Nippon, etc.)
- 🔮 Future price prediction (next 10 days)
- 📉 Model evaluation using RMSE
- ⚡ Instant updates (no button required)

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas, NumPy
- Matplotlib
- Yahoo Finance API

## ▶️ How to Run
```bash
pip install -r requirements.txt
python -m streamlit run app.py
